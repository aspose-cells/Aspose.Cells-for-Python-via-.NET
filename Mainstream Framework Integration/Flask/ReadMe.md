# Flask

[Flask](https://flask.palletsprojects.com/) is a lightweight and flexible Python web framework for building websites and web applications. Unlike Django, Flask is minimalistic, giving developers more control and freedom to structure their projects. It is highly extensible and widely used in both small and large web projects.

You can find the source code and contribute to Flask on GitHub: [https://github.com/pallets/flask](https://github.com/pallets/flask)

This example demonstrates how to integrate [Aspose.Cells for Python](https://pypi.org/project/aspose-cells-python/) into a **Flask** project to generate and export Excel reports dynamically.

Run the project:
python app.py


Access:
http://127.0.0.1:5000/

Click "Download Sales Report," and a file named sales_report.xlsx will be automatically generated, containing:

Column headers (bold)

5 rows of sample data

A column chart
